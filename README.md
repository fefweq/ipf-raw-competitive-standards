# IPF Raw Competitive Standards FYP

## Title
Are Competitive Standards Rising? Performance Trends and Winning Thresholds in IPF Raw Powerlifting Using OpenPowerlifting

## Data Scope
- Data source: OpenPowerlifting
- Federation: IPF only
- Equipment: Raw only
- Event: SBD / Full Power only
- Time window: 2009–2024
- Sex: M and F only
- Main outcome: TotalKg
- Secondary outcome: Win (champion vs non-champion)

## Project Goal
This project examines whether competitive standards in IPF Raw full-power powerlifting increased between 2009 and 2024 using the OpenPowerlifting database. It focuses on performance trends, upper-level performance thresholds, and factors associated with winning.

## Objectives
- Describe participation structure and trends in IPF Raw full-power competitions
- Analyze temporal trends in TotalKg
- Evaluate whether upper-level competitive thresholds increased over time
- Compare trends across sex and weight class
- Identify factors associated with winning
- Use participation growth as contextual evidence for possible increases in competitive intensity
